# active-directory-homelab
A hands-on Active Directory home lab built using VirtualBox to simulate a real-world enterprise environment. This project demonstrates domain setup, user and group management, Group Policy configuration, and basic security monitoring using Event Viewer.

* Architecture - 

Domain Controller: Windows Server
Client Machine: Windows 10/11
Network: Internal Virtual Network

* Key Features - 

Active Directory Domain Services (AD DS)
DNS configuration
User & Group Management
Group Policy (GPO)
Event Viewer log analysis

* Security Use Cases - 

Monitoring failed login attempts (Event ID 4625)
Tracking successful logins (Event ID 4624)
Tracking Kerberos failure (Event ID 4771)
Account lockout detection
